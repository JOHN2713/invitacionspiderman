# 🕷️ Invitación Spider-Man

Una invitación interactiva con temática de Spider-Man para ir al cine.

## ✨ Características

- **Paso 1**: Invitación al cine con efecto especial en el botón "No"
- **Paso 2**: Selección de color de hoddie (Rojo o Negro)
- **Paso 3**: Selección de fecha disponible
- **Paso 4**: Confirmación con ingreso de correo electrónico

## 🎨 Efectos Especiales

- Telarañas animadas en el fondo
- Efecto "glitch" en los títulos
- Confeti al aceptar la invitación
- Fuegos artificiales en la confirmación
- Corazones cayendo en la pantalla final
- Partículas brillantes al mover el mouse

## 🚀 Cómo usar

1. Clona el repositorio
2. Abre `index.html` en tu navegador
3. ¡Disfruta la experiencia!

## 📁 Archivos

- `index.html` - Estructura HTML
- `style.css` - Estilos y animaciones
- `script.js` - Lógica interactiva
- `spidermanhoddie.png` - Imagen del hoddie
